## Functional setState 
The function returned for updating state can also take a function similar to the good ol' `setState`. 

```js
 const [value, updateValue] = useState(0)
 // both forms below are valid ðŸ‘‡
 updateValue(1)
 updateValue(previousValue => previousValue + 1)
```
This is ideal when the state update depends on some previous value of state.
