# useContext ðŸ‘œ

`useContext` saves you the stress of having to rely on a Context consumer e.g. 
MyContext.Consumer and the render props API it exposes

```js
// before: 
const ThemeContext = React.createContext("dark");

function Button() {
    return <ThemeContext.Consumer>
        {theme => <button className={theme}>Amazing button</button>}
    </ThemeContext.Consumer>
}
```

```js
// now: 
import {useContext} from 'react';

function ButtonHooks() {
  const theme = useContext(ThemeContext)
  return <button className={theme}>Amazing button</button>
}
```
